## How to run

```
pip install -r requirements.txt
export OPENAI_API_KEY=<you-openai-api-key> # you can put it to .bashrc/.zshrc
```

#### Web UI
```
streamlit run main_web.py
```
![web_exmaple](./assets/web_example.jpg)

#### CLI UI:

```
python main_cli.py
```
